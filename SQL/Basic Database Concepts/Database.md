A database is an organized collection of data that can be easily accessed, managed, and updated. It allows for efficient data retrieval and manipulation.

Common Types of Databases:
Relational Databases (RDBMS): Store data in tables with rows and columns. Examples include MySQL, PostgreSQL, and Oracle.

NoSQL Databases: Designed for unstructured or semi-structured data. Types include:

1. Document-based (e.g., MongoDB)

2. Key-Value Stores (e.g., Redis)

3. Column-Family Stores (e.g., Cassandra)

4. Graph Databases (e.g., Neo4j)
